function _decode_values(values, conversors) returns Decoded_Values 

Decoded_Values = [None if value is None else conversor(value)
                      for conversor, value
                      in zip(conversors, values)]
zip(conversors, values) is an array of tuples generated from items in the lists conversors and values:

Zipped_Converser_Values =
zip(conversors, values) = ((conversors [1], values[1]),
			   (conversors [2], values[2]),
		           etc.);

for conversor, value in zip(conversors, values) means
for index in Zipped_Converer_Values (first) to Zipped_Converer_Values (last)
	conversor = Zipped_Converer_Values (index, converser)
	value = Zipped_Converer_Values (index, value)

	if conversor is not none
		Append conversor (value) to Decoded_Values

Example:
_decode_attribute(self, s)  decodes an attribute line.

Attributes follow the template:  <attribute-name> <datatype>
where:
 attribute-name is a string
datatype can be:
 A Numerical attribute such as NUMERIC, INTEGER or REAL.
 A String as STRING.
 A Nominal attributes with format:
            {<nominal-name1>, <nominal-name2>, <nominal-name3>, ...}


A convertor is initialised with tuple (ATTRIBUTE_NAME, TYPE_OR_VALUES)
